Installation Guide

Because Colit is an open-source tool, Windows might be extra cautious when you install it. Here is how to get started in seconds.

"Windows protected your PC"

You might see a blue window popup saying Windows SmartScreen prevented an unrecognized app from starting.

Solution:

  1. Click the underlined "More info" text.
  2. A new button will appear at the bottom.
  3. Click "Run anyway".

Installation Error

If the app doesn't open or shows missing file errors, you might be trying to run it inside the ZIP file.

Solution:

  1. Right-click the colit_installer.zip file.
  2. Select "Extract All...".
  3. Open the extracted folder and run Colit.exe.
